# 包管理工具

  • npm：Node Package Manager，是JavaScript的包管理器。它允许你安装、共享和重用代码片段。npm的官方网站是https://www.npmjs.com/ (opens new window)。

  • yarn：是一个替代于npm的工具，由Facebook, Google, Exponent 和 Tilde 共同创建。它的设计目标是更快、更可靠地解决依赖关系问题。yarn的官方网站是https://yarnpkg.com/ (opens new window)。

  • pnpm：是一个快速的、节省磁盘空间的包管理器。它通过创建一个共享的全局锁文件来工作，这样可以减少磁盘使用并提高安装速度。pnpm 的官方网站是 https://pnpm.io/ (opens new window)。

  • cnpm：是一个 npm 的淘宝镜像，主要用于解决在中国大陆地区访问npm官方网站速度慢的问题。cnpm的官方网站是https://npmmirror.com/ (opens new window)。

  • npx：是一个 npm 包执行工具，允许你运行安装在本地、全局或远程的npm包的命令。npx 的官方文档可以在 [https://docs.npmjs.com/cli/v7/commands/npx](https://docs.npmjs.com/

  • nrm ：是一个 npm 镜像源管理器，允许你快速切换不同的npm注册表。nrm 的官方网站是 https://github.com/Pana/nrm (opens new window)。

# 工程化工具

  • webpack：一个现代 JavaScript 应用程序的静态模块打包器(module bundler)。当 webpack 处理应用程序时，它会递归地构建一个依赖关系图，其中包含应用程序需要的每个模块，然后将所有这些模块打包成一个或多个 bundle。webpack 的官方网站是 https://webpack.js.org/ (opens new window)。
