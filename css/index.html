# css 清除浮动

1. 额外标签法（不推荐）
2. 父级添加overflow属性值hidden或auto。
3. 使用after伪元素清除浮动。
4. 使用双伪元素清除浮动。
5. 创建公共类clearfix，使用zoom:1触发haslayout(IE6/7)。
6. 使用flex布局。
7. 使用grid布局。
8. 使用BFC布局。
9. 使用table布局。
10. 使用margin负值法。
11. 使用浮动本身。
12. 使用绝对定位。
13. clear: both
